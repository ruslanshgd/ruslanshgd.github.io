- Разработала дизайн для сервиса рассылки смс-сообщений абонентам. Это ускорило запуск новых смс кампаний в несколько раз по сравнению со старым решением.
- Разрабатывала дизайн для облачной платформы, где сотрудник компании может получить инфраструктуру для своего продукта.
- Обучала стажёра, который потом успешно перешёл на позицию младшего дизайнера.
- Нанимала дизайнеров, проводила дейли, 1/1, дизайн-ревью.
